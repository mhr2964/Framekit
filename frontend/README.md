# Framekit Frontend

Frontend scaffold for **Framekit**.

- Product: Framekit helps freelancers and small teams collect polished, timestamped website feedback in elegant review rooms that clients actually enjoy using.
- Repository: https://github.com/mhr2964/Framekit.git

## Purpose

This workspace gives the frontend team an immediate handoff-ready app scaffold with TypeScript, Next.js App Router, linting, formatting, and a minimal file layout that supports parallel delivery before backend readiness.
