### Labels
- **Page title:** Create a room
- **Room name label:** Room name
- **Room name placeholder:** Q2 homepage review
- **Frames label:** Frames
- **Frames helper:** Add the screens you want feedback on.
- **Review note label:** Review note
- **Review note placeholder:** Tell reviewers what to look for.
- **Invite reviewers label:** Invite reviewers
- **Invite reviewers placeholder:** name@company.com
- **Share link label:** Share link
- **Optional label:** Optional

### Actions
- **Primary CTA:** Create room
- **Secondary CTA:** Cancel
- **Add frame CTA:** Add frame
- **Upload file CTA:** Upload file
- **Paste link CTA:** Paste link
- **Add another CTA:** Add another
- **Remove action:** Remove
- **Copy link CTA:** Copy link
- **Send invites CTA:** Send invites
- **Close action:** Close
- **Back action:** Back

- **Room name helper:** Use a clear name reviewers will recognize.
- **Frames upload helper:** PNG, JPG, or PDF.
- **Review note helper:** Keep the brief short and specific.
- **Invite helper:** Invite now or share the link later.

### Empty states
- **No frames added:** No frames added yet.
- **No reviewers added:** No reviewers added yet.
- **No review note:** No review note added.
- **Empty review state:** No feedback yet.

### In-progress states
- **Creating room:** Creating room...
- **Uploading frames:** Uploading frames...
- **Sending invites:** Sending invites...

- **Posting comment:** Posting comment...

- **Room created:** Room created.
- **Frames uploaded:** Frames uploaded.
- **Invites sent:** Invites sent.
- **Link copied:** Link copied.
- **Review submitted:** Review submitted.
- **Comment posted:** Comment posted.

- **Create room error:** We couldn’t create the room. Try again.
- **Upload error:** We couldn’t upload that file. Try again.
- **Invite error:** We couldn’t send invites. Try again.
- **Copy link error:** We couldn’t copy the link.
- **Submit review error:** We couldn’t submit your review. Try again.
- **Comment error:** We couldn’t post that comment. Try again.

### Invite and share
- **Invite section title:** Invite people to review
- **Share section title:** Or share a link
- **Share helper:** Anyone with the link can review.
- **Invite success note:** Reviewers are on the way.
- **Share success note:** Link ready to share.
