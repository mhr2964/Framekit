# Platform Handoff Notes

This directory is reserved for platform-owned coordination artifacts.

## Scope

Platform supports:
- workspace structure
- shared folder conventions
- contract documentation scaffolding
- repo-level coordination notes

Platform does not own frontend/backend product implementation.

## Current baseline

- Frontend baseline exists in `workspace/frontend/`
- Backend baseline exists in `workspace/backend/`
- Shared scaffold and canonical contracts path are now present for parallel engineering

## When to use this folder

Add docs here when a repo-wide convention, integration note, or handoff instruction does not belong to a single product department.
